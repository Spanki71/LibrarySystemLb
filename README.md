# Информационная система «Библиотека»

## Описание проекта
Консольное приложение для управления библиотечным фондом, читателями и операциями выдачи/возврата книг.

## Функциональные возможности
- Добавление новых книг в каталог
- Регистрация читателей
- Выдача книг читателям
- Возврат книг
- Просмотр каталога книг
- Просмотр активных выдач
- История выдач читателя

## Технологии
- Язык программирования: Python 3
- Система контроля версий: Git
- Хостинг: GitHub
